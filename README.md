# Kursi-online

Kjo është dokumentacioni i projektit "Kursi-online".

## Përshkrimi
Ky projekt krijon një platformë online për menaxhimin e kurseve. Aplikacioni ofron mundësinë për të parë kurset, regjistruar përdoruesit dhe ndjekur informacionin e përshtatshëm.

## Funksionalitetet kryesore
- Shfaqja e kurseve në dispozicion
- Regjistrimi i përdoruesve
- Menaxhimi i përmbajtjes së kurseve

## Si të përdoret
1. Klono depozitën
2. Instalo varësitë
3. Përfshi skedarët e projektit

## Gjendja aktuale
Ky dokument është përditësuar sipas përmbajtjes së projektit dhe ndryshimeve të bëra në këtë dosje.
